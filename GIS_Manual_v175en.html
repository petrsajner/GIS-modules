1 · Quick Start

  1. Open google-image-studio_v175en.html in Chrome.
  2. Click ⚙ Setup in the top menu. Paste your Google AI key (required) and fal.ai key (for FLUX, Kling, video, upscale). See the separate API Setup Guide for where to get keys.
  3. Go back to ⚡ Generate. Select a model, write a prompt, and click ▶ Generate or press Ctrl+Enter.
  4. Your image appears in the center area. Hover over it for download, upscale, reuse, and other options.

✦ AI Prompt Tool

Button next to the prompt field. Opens an 820px panel — badge shows [IMAGE] or [VIDEO] depending on current mode. Five tabs:

TabWhat it does
✦ EnhanceGemini expands a rough idea into a detailed cinematic prompt
◈ VariantsGenerates 2–6 alternative versions — displayed as clickable gold-bordered cards. Click any card to send it to the result field.
⇄ TranslateTranslates any language to English without changing style or content
🎲 RandomSelect genre chips (or none = anything) and generate a surprising photographic/cinematic prompt
💬 ChatMulti-turn conversation for iterative prompt development

Input / Output layout (all tabs except Random and Chat)

  • Green field (top) — Input prompt. Pre-filled from the current buffer when opening or switching tabs.
  • "↓ Use as result" button (top right) — copies input directly to the result field without running AI. Use this if no variant satisfies you.
  • Red field (bottom) — Result. Empty on open, populated after running. Editable.
  • Switching tabs clears the result field — you always see the result of the current tab only.
  • ↗ Use as Prompt (footer) — writes the current result to the main prompt field and closes the panel.

Chat tab details

  • On open: a grey "Working with prompt:" context bubble shows the current prompt at the top. The input field is empty — just type your editing instruction (e.g. "add fog and make the mood darker").
  • Each AI reply becomes the new working version of the prompt. The chat preserves all prior content — it only changes what you ask for.
  • ↺ New chat — resets conversation and shows a fresh context bubble with the current buffer
  • 🎲 Random start — generates a random concept and places it in the input field
  • The last AI reply is automatically placed in the buffer for ↗ Use as Prompt

7 · Reference Images

The reference panel appears below the prompt field when the selected model supports references. References are links to assets in the Asset Library — original image data is stored once there, never duplicated.

Adding references

＋ tile → upload from file ☰ Assets → pick from library ⊕ button on result card Drag from Assets
  • Uploading saves to Assets automatically — no duplicate storage
  • Max references per model varies (6–14 for image, 1–7 for video)

Renaming references

  • Double-click label under thumbnail → inline text input → Enter to confirm, Escape to cancel
  • Renaming updates the Asset Library and all active refs sharing the same asset

Other actions

  • Drag to reorder — changes which ref is sent first
  • ✦ Describe (hover) — Gemini analysis: AI Prompt or brief description (see section 5)
  • × (hover) — removes from active panel (does not delete from Assets)
  • Click thumbnail — opens full-size preview lightbox

Max 2048px on send

Checkbox in the ref panel (default: off). When on, images are resized to max 2048px before the API call. Originals stay at full resolution in Assets.

16 · Tips & Workflows

Combining styles

Select multiple styles at once — they merge into a single combined instruction. "Golden Hour" + "Wes Anderson" + "Oil Painting" all work together. Select a Camera position for consistent framing.

Reuse workflow

Generate → click ↺ Reuse → adjust prompt or one parameter → generate again. Reference images are also restored. Use a fixed seed for reproducible variations (seed, seed+1, seed+2…).

Describe → Prompt

Upload a reference → hover → ✦ Describe → select AI Prompt tab → click ↳ Use as prompt. Edit as needed and generate. Great for recreating the style of an existing image.

Chat — iterative prompt development

Generate → ↺ Reuse → open AI Prompt → Chat tab. The current prompt is shown in the grey context bubble. Type only your instruction: "add fog and make the mood darker" — Gemini builds on the existing prompt. Keep refining until satisfied, then click ↗ Use as Prompt and generate.

Upscale workflow

Generate at 1K or 512 to quickly check composition. Once satisfied → ⬆ Upscale. Recraft Crisp for photos, Clarity or Topaz Gigapixel for AI art, SeedVR2 for faces and textures.

Parallel variants — fixed seed

Set count to 4 and a fixed seed. FLUX and Z-Image use seed, seed+1, seed+2, seed+3 — four reproducible variations side by side. Compare and pick the best.

Qwen / WAN Edit instructions

Write naturally: "Remove the person on the left" · "Change background to sunset over ocean, keep subject sharp" · "Add text SALE in bold red at the top". Upload the image to edit as a reference.

Folder workflow

Set the Target folder dropdown (above the generate button) before generating — all results from that session go directly into the folder. No manual moving needed. The folder marker (blue bar) is visible in the Gallery for items assigned to a folder.

Upload for upscale

Have an image from another tool? Use ↑ Upload in the Gallery, then ⬆ Upscale. Works for any external image.

Video from gallery image

In the gallery, hover over an image → ⊕ Ref & Assets — in Video mode the image is added directly as a video reference (start frame). Switch to the Video tab and generate.

Wan R2V Flash — consistent characters

Upload up to 5 character photos as video refs. Reference them by name in the prompt: "Character1 walks through a market while Character2 watches from a balcony."

Batch processing

Submit 4–8 generate jobs quickly one after another — they run in parallel via the async queue. Results appear as they complete, no need to wait between submissions.
