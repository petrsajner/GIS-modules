How to get your API key
  1. Open your browser and go to:
    https://aistudio.google.com/apikey
  2. Sign in with your Google account if asked.
  3. Click the blue button "Create API key".
  4. A long string of letters and numbers appears — that is your key. Click the copy icon next to it (or select all and press Ctrl+C).
→ Enter this key in Generative Image Studio
  1. Open GIS in Chrome.
  2. Click ⚙ Setup in the top menu.
  3. Find the field labelled "Google API Key".
  4. Click inside it and paste your key (Ctrl+V).
  5. The dot next to the field turns green — you're done.
The same key also appears in the top-right header bar for quick access during generation.

How to get your API key
  1. Go to:
    https://fal.ai/dashboard/keys
  2. Create a free account or sign in (Google sign-in available).
  3. Add a payment method first — fal.ai requires it before generating keys.
    Click Billing in the left sidebar → "Add payment method". You are only charged for what you use.
  4. Go back to Keys in the sidebar. Click "Add key", give it any name (e.g. "GIS"), then click "Create".
  5. Your key appears once — copy it immediately (Ctrl+C). It won't be shown again.
→ Enter this key in Generative Image Studio
  1. Click ⚙ Setup in the top menu.
  2. Find the field labelled "fal.ai API Key".
  3. Click inside it and paste your key (Ctrl+V).
  4. The dot next to the field turns green — you're done.

3 · Replicate — WAN 2.7 Image

Optional ~$0.03/image · requires Proxy
Image generation & editing
WAN 2.7 WAN 2.7 Pro WAN 2.7 Edit WAN 2.7 Pro Edit
Requires Proxy: WAN 2.7 image models run via the GIS proxy worker due to Replicate CORS restrictions. The Proxy URL must be configured in Setup before these models become available.
How to get your API key
  1. Go to:
    https://replicate.com/account/api-tokens
  2. Create a free Replicate account or sign in.
  3. Add a payment method first — Replicate requires it before using paid models.
    Go to Billing in the sidebar and add a payment method.
  4. Click "Create token", give it a name, and copy the key. It starts with r8_.
→ Enter this key in Generative Image Studio
  1. Click ⚙ Setup in the top menu.
  2. Find the field labelled "Replicate API Key".
  3. Paste your key and the dot turns green.
  4. Also ensure the Proxy URL field is set.

How to get your API key
  1. Go to:
    https://console.x.ai
  2. Sign in with your X (Twitter) account or create a new xAI account.

  4. Name it anything, then click "Create". Copy the key that appears.
→ Enter this key in Generative Image Studio
  1. Click ⚙ Setup in the top menu.
  2. Find the field labelled "xAI API Key".
  3. Paste your key and the dot turns green.

How to get your API key
  1. Go to:
    https://www.topazlabs.com/account
  2. Sign in or create a Topaz Labs account.
  3. Navigate to API or Developer section in your account dashboard. Click "Generate API Key".
  4. Copy the key that appears.
→ Enter this key in Generative Image Studio
  1. Click ⚙ Setup in the top menu.
  2. Find the field labelled "Topaz API Key".
  3. Paste your key and the dot turns green.
All keys are saved automatically.
You only need to enter them once. They are stored locally in your browser and will be there every time you open the program. They are never sent anywhere except directly to the API service you chose. Use ⚙ Setup → API Keys Backup to export all keys at once for safe transfer to another computer.
